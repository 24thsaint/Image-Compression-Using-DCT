## Getting Started

1. Make sure you have [NodeJS](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.
2. Install the dependencies

    ```
    cd path/to/project; npm install
    ```

3. Start the app

    ```
    npm start
    ```
